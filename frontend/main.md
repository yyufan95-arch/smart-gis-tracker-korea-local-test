# RealTrack Tour - 주요 프로젝트 설명
## 1. 프로젝트 목표
본 시스템은 GPS와 GIS를 활용하여 관광객의 실시간 위치를 추적하고, 지정된 미션 지점에 도착하면 자동으로 인증을 완료하는 스마트 관광 안내 시스템을 구현하는 것을 목표로 합니다.

## 2. 시스템 구조
프로젝트는 다음과 같은 모듈로 구성되어 있습니다:
-   **frontend**: 사용자 인터페이스 및 지도, GPS 기반의 프론트엔드 로직
-   **service**: 인증 처리 및 데이터 저장 등의 백엔드 비즈니스 로직
-   **web**: 프론트엔드와 백엔드를 연결하는 API 인터페이스
-   **database**: 인증 기록 및 사용자 데이터 관리

## 3. 주요 기능 상세
### 3.1 실시간 위치 추적
브라우저 GPS를 활용하여 사용자의 실시간 위치를 지도에 표시합니다.

### 3.2 근접 자동 인증
사용자가 목표 지점 5m 이내에 도착하면 자동으로 인증이 완료됩니다.

### 3.3 인증 기록 관리
인증 완료 시각, 단계, 위치 정보가 데이터베이스에 저장되며, 프론트엔드에서 실시간으로 확인할 수 있습니다.

## 4. 테스트 버전 안내
본 프로젝트에는 두 가지 테스트 버전이 포함되어 있습니다:
-   **인도네시아 버전**: 초기 기능 검증용 고정 좌표 테스트
-   **현지 테스트 버전**: 실제 환경에서의 작동 확인을 위한 로컬 좌표 버전

## 5. 향후 개선 방향
-   백엔드 데이터베이스 연동 강화
-   다중 사용자 인증 지원
-   오프라인 모드 기능 추가
